Pertanyaan Sering Diajukan (FAQ)

Jawaban untuk pertanyaan yang mungkin kamu miliki.

Apakah alat ini aman digunakan?

Ya, sangat aman. Semua pemrosesan data terjadi secara lokal di browser Anda menggunakan teknologi klien (client-side). Data Instagram Anda tidak pernah meninggalkan perangkat Anda, tidak menyentuh server kami, dan tidak disimpan online. Privasi adalah prioritas utama kami.

Apakah saya perlu login akun Instagram di sini?

Tidak. Anda tidak perlu memberikan password atau login Instagram. Anda hanya perlu mengunduh file data resmi dari pengaturan Instagram Anda sendiri, lalu mengunggah file JSON tersebut ke sini.

Kenapa harus pakai format JSON?

Format JSON adalah format data yang mudah dibaca oleh sistem komputer. Format HTML ditujukan untuk dibaca manusia (tampilan web). Agar alat kami bisa menghitung unfollowers secara otomatis dan akurat, kami memerlukan struktur data dari file JSON.

Berapa lama proses download data dari IG?

Biasanya Instagram menyiapkan data dalam 5-10 menit jika Anda hanya memilih "Pengikut dan Mengikuti" (karena hanya teks, file kecil). Namun, jika Anda memilih "Semua Informasi" termasuk foto/video, bisa memakan waktu hingga 48 jam.
